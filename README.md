## 游戏简介

**游戏标题：** TWT

**TWT背景设定：**
TWT 是一个SCP基金会与相关组织临时组建的战术歼灭部队，由前MTF精英成员和部分合作组织成员组成，负责处理高危事件。

**游戏玩法：**
该游戏为一款2D像素肉鸽射击游戏，游戏中玩家将扮演一名TWT干员，在随机地图(由不同的可复用区块随机拼凑而成)中进行战斗，通过射击、躲避、逃跑等多种方式与敌人进行战斗。游戏中还会有一些其他的玩法，例如：
- 收集物品
- 制作装备
- 解锁技能
- 探索地图

**游戏背景故事：**
TWT-[数据删除]编队在临时调派site-09站点执行[数据删除]任务时，发生了大规模的现实扭曲事件，站点时间空间结构被打乱(对应游戏的随机地图生成)，而游戏操控角色(以下简称"你")幸存了下来，你需要在站点的自毁协议启用前逃离该站点。

**玩家角色设定：**
作为TWT高级特遣人员，你拥有的应急反应装置会在你每次死亡后重置世界线(即复活)，将你重置到初始房间，重新开始探索。

**操作说明：**
- `wasd` 键控制角色移动(地图渲染区域会随角色的位置移动保证角色处于画面中央)
- 鼠标左键射击
- 鼠标右键(未定)
- 鼠标指针相对控制角色的方向为射击与视线方向
- `r` 键换弹
- `空格键` 冲刺躲避
- `f` 键为通用交互键
- `esc` 键退出游戏

**游戏目标：**
活下去，逃出去
